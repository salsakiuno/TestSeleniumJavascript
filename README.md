# TestSeleniumJavascript
Selenium WebDriver using node, mocha and chia

# How to install 
clone the repo
# Install libraries 

- selenium-webdriver
- mocha
- chia
- chia-promised

# How to run 
npm test test/amazon.test.js

# Project structure

- lib -> Constains the PageObject
- locators -> Contain the locators
- test -> contain the test
